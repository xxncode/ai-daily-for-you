命中注定 · 稳稳接住你

每天早上，一份为你一个人排的 AI 行业报纸。

别人的简报都在回答"今天行业发生了什么"。这份报纸多一句——"这对你意味着什么"：每条新闻末尾，结合你的身份、领域、目标，落到一个具体的机会或动作。

▶ 看一份示例样张

两个不一样

像报纸，不像网页。衬线字体、模块化互锁版式、真实原文配图——翻它像翻一版报纸，读着不累。

每条落到"对你"。为你一个人调；换个人、换份 profile，报纸就变成对他的。

怎么变成你的

  1. 填一份 profile：你是谁、你的领域、目标、关注方向、报名。
  2. 每天把当天新闻 + 生成规范 + 你的 profile 交给任意一个能联网的 AI。
  3. 它产出当天属于你的那份报纸。

方法和模板都在 GitHub 仓库里（PROMPT.md + profile.example.md），欢迎 fork 成你自己的。
