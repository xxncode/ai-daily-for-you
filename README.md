# 命中注定能够稳稳接住你的专属 AI 日报

> 每天早上，一份**为你一个人**排的 AI 行业报纸。
> 别人的简报是"今天发生了什么"；这份多一句——**"这对你意味着什么"**。

[**▶ 看一份示例样张**](./sample.html)

---

## 它跟别的 AI 简报有什么不一样

市面上的 AI 简报无穷多，它们都在回答同一个问题："今天行业发生了什么。" 这份报纸多做两件事：

1. **像报纸，不像网页。** 衬线字体、模块化互锁版式（俄罗斯方块式的咬合，不是一条条往下刷）、真实原文配图。翻它像翻一版报纸，读着不累。
2. **每条落到"对你"。** 每条新闻末尾，结合你自己的身份、领域、目标，给一句"所以对你来说，这意味着什么 / 能反推什么机会"。它是为你一个人调的——换个人，换份 profile，报纸就变成对他的。

## 怎么做到的（三层结构）

| 文件 | 是什么 |
|---|---|
| `sample.html` | 一份完整样张 = 固定 CSS + 版式模板（**模板层**，锁死不动） |
| `PROMPT.md` | 生成规范：怎么选头条、怎么写洞察/对你、图片怎么处理、版式怎么排（**规范层**，跨模型通用） |
| `profile.example.md` | 读者是谁、关注什么、报名叫什么（**配置层**，每个人自己填） |

**为什么这样拆**：版式的整齐、腔调的一致、图片的诚实——都靠固定的模板和明确的规范锁住，不靠模型每天即兴发挥。换任何一个够强的模型（Claude / GPT / Gemini / Qwen / DeepSeek），产出都八九不离十。

## 怎么变成"你的"

1. 复制 `profile.example.md` → `profile.md`，填上你是谁、你的领域、你的目标、你关注的方向、你想给报纸起的名字。
2. 每天把「当天的 AI 新闻素材」+ `PROMPT.md` + 你的 `profile.md` + `sample.html` 一起交给一个能联网、能写文件的 AI。
3. 它就产出当天属于你的那份报纸。

## 关于这份示例

`sample.html` 里的"对你"，是针对一个**虚构读者**（东京做跨境电商的创业者）生成的，用来演示"对你"这个功能长什么样。换成你的 profile，每一句都会变。
